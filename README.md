
# TravelSafe – Final Professional Project

Full‑stack tourist safety & travel planner.

Stack
- React (Vite)
- Node.js + Express
- MongoDB
- JWT Authentication

Major Features
- Authentication (Signup/Login)
- Tourist Destinations Explorer
- Search & Filter
- Trip Planner + Itinerary
- Expense Tracker with categories
- Safety Center (SOS)
- Weather Monitoring
- Disaster Alerts
- Nearby Attractions (Maps Ready)
- Hidden Gems suggestions
- Translation helper (structure)
- Scam alerts system
- Mobile responsive UI
